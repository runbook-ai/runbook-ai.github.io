2. Description of Service

Runbook AI is a browser based automation framework. It allows users to:

  • Connect their Gmail account to a local browser environment via an OAuth-enabled web gateway.
  • Automate browser-based tasks ("Runbooks") using user-provided Large Language Model (LLM) APIs.
  • Execute autonomous agent flows triggered by incoming emails.
